# 🚀 NavLogic — Smart Route Finder

NavLogic is a full-stack route optimization application that uses **Dijkstra’s Algorithm** to calculate and compare multiple paths between cities and determine the shortest route with interactive map visualization.

---

## 🌟 Features

- 📍 Find shortest path between cities
- 🧠 Dijkstra’s Algorithm implementation
- 🔁 Compare multiple possible routes
- 🗺️ Interactive map visualization using Google Maps
- 📊 Displays distance for each path
- 🎯 Highlights optimal (shortest) route

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Google Maps API

### Backend
- Node.js
- Express.js

### Concepts Used
- Graph Data Structure
- Dijkstra’s Algorithm
- Haversine Formula (distance calculation)
